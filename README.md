# Display all users, add new user, and show user by ID
Technologies used: 
    Node and Express for the backend
    Ionic, Angular, and TypeScript for the frontend

## Demo
[!Demo](demo.gif)

## Install
```
git clone https://github.com/OumaimaTiguint/users.git
```

```
npm install
```

## Start the server
```
cd server
```

```
nodemon index.js
```

## Run the app

```
ionic serve
```
